## 📂 Featured Projects

### 1. 🤖 Expert Corporate Intelligence Agent (RAG)
![Status](https://img.shields.io/badge/Status-Completed-success)
![Tech](https://img.shields.io/badge/Focus-Enterprise_AI-blueviolet)

> *An enterprise-grade system that allows employees to query internal financial documents with "Zero-Hallucination" accuracy.*

* **Problem:** Generic OCR fails to read financial tables (confusing '8' for '3'), and LLMs hallucinate numbers.
* **Solution:** Built a "Vision-First" pipeline using **Gemini 2.5 Pro** and **Pinecone** with a strict "Zero-Guessing Protocol."
* **Key Tech:** Hybrid Search (Dense + Sparse), Cross-Encoder Reranking, Docker.
